# 🚀 CodeSpeak AI

CodeSpeak AI is a voice-based coding assistant designed to help visually impaired users write, understand, and execute code using speech.

---

## 🎯 Features

* 🎤 **Voice Input**
  Speak your coding instructions instead of typing.

* 🧠 **Code Generation**
  Converts voice commands into working code.

* ⚡ **Code Execution**
  Runs the generated code instantly.

* 📖 **Code Explanation**
  Explains code in simple and easy-to-understand language.

* 🔊 **Text-to-Speech**
  Reads explanations aloud for better accessibility.

---

## 🛠 Tech Stack

* **Frontend:** React + TypeScript (Figma-based UI)
* **Backend:** Spring Boot (Java)
* **Voice Input:** Web Speech API
* **Code Execution:** Java Compiler API

---

## 🚀 How to Run the Project

### 🔹 1. Backend Setup

```bash
cd backend
mvn spring-boot:run
```

Backend will run on:

```
http://localhost:9090
```

---

### 🔹 2. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend will run on:

```
http://localhost:5173
```

---

## 🧪 How to Use

1. Open the frontend in browser
2. Click the 🎤 microphone button
3. Speak your coding requirement (e.g., *“print even numbers till 20”*)
4. Click:

   * Generate → to generate code
   * Run → to execute code
   * Explain → to understand code
5. Listen to explanation using 🔊
